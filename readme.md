# readme.md

本文件为 Claude Code (claude.ai/code) 在本项目中工作提供指导。

## 项目概述

这是一个用于生成 LeetCode 算法题解交互式 HTML 可视化的 Claude Code skill。生成的独立 HTML 文件可逐步演示算法执行过程。

## 核心组件

### Skill 定义
- **位置**: `.claude/skills/gen-solution-viz/SKILL.md`
- **用途**: 为 LeetCode 题目生成交互式 HTML 可视化
- **触发关键词**: "生成题解可视化", "创建一个可视化页面", "帮我写一个算法演示", /gen-viz

### 设计规范
- **位置**: `.claude/skills/gen-solution-viz/design-guide.md`
- **用途**: 定义生成 HTML 的视觉设计标准
- **关键规格**: 深色主题 (#0f0f23 → #1a1a2e)、VS Code Dark+ 代码高亮、CSS 变量主题

## 生成 HTML 的要求

生成可视化时需遵循以下关键要求：

### 布局结构
- 使用 `vh` 单位实现固定高度布局 - 页面必须占满视口，不允许滚动
- 左侧面板 (45%)：代码展示区，带独立滚动条
- 右侧面板 (55%)：可视化区域（固定，不滚动）
- 顶部：题目名称和算法类型（居中）
- 底部：键盘快捷键提示

### 代码高亮
- 每行代码用 `<span class="code-line" data-line="N">` 包裹
- 当前行高亮背景色 `rgba(99, 102, 241, 0.25)`，左侧有颜色边框
- **关键**：高亮滚动到代码块中央（不是整个页面）

### 变量状态面板
- 步骤变化时变量实时更新
- 变量名按类型区分颜色：
  - 指针/索引变量 (i, j, left, right)：橙色 (#f59e0b)
  - 数值变量 (sum, ans, max)：紫色 (#818cf8)
  - 布尔变量：绿色 (#10b981)
  - 对象变量：橙色 (#fb923c)

### 动画功能
- 开始/暂停、上一步/下一步、重置、自动播放按钮
- 平滑过渡效果 (0.3s cubic-bezier)
- 键盘快捷键（←/→ 切换步骤，Space 播放/暂停）

### 参数输入
- 用户可自定义算法参数
- 支持格式：数组 `[1,2,3]`、字符串 `"hello"`、数字 `123`

## 输出位置

生成的 HTML 文件保存至：
```
/nginx/html/solutions/<title-slug>-<timestamp>.html
```

写入文件前需确保目录已存在。
